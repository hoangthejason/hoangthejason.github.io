A vector space is best understood by asking which operations remain valid inside the set.

## Closure

A set is a vector space only if addition and scalar multiplication keep you inside the same set.

## Span

The **span** of a set of vectors is the collection of all linear combinations of those vectors.

## Basis

A basis should do two things at once:

1. generate the space
2. avoid redundancy

That is why linear independence matters.
